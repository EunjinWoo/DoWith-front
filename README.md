# DoWith

### 🧐 프로젝트 소개
그룹 스터디를 효율적으로 관리하기 위한 서비스입니다.
기존에 친구들과 카카오톡으로 매주 인증하며 진행하던 스터디에서 착안하여, 체계적으로 스터디 계획을 관리하고 인증할 수 있도록 하는 것에 중점을 둔 application을 만들고자 했습니다.
</br>

### 🧞‍♂️ 핵심 기능
- 스터디 그룹 및 계획 조회
- 멤버 별 진행도 확인
- 설정한 목표 별 인증 기능
- 목표 설정
- 멤버 초대
</br>

### 📚 기술 스택
Dart, Flutter
